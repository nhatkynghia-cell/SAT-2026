▶ Việc của tôi — làm song song khi Claude code

Checklist bấm-là-xong. Hầu hết chỉ cần copy "chìa khóa" gửi Claude; phần ráp code + kiểm thử Claude lo. Chi tiết đầy đủ: file HUONG_DAN_USER.md (mở bằng MoHuongDan.bat).

Ưu tiên 1 (KHUYÊN DÙNG VN)

🏦 payOS — chuyển khoản QR (cá nhân, CCCD)

  1. Vào payos.vn → đăng ký bằng CCCD (~5 phút, không hợp đồng).
  2. Liên kết TK ngân hàng cá nhân (MB/OCB/ACB/BIDV) — tên khớp tên đăng ký.
  3. Lấy 3 chìa khóa: Client ID + API Key + Checksum Key → gửi Claude.
  4. Bật webhook trong dashboard payOS: URL https://sat-2026.vercel.app/api/payment/payos-webhook.
  5. Supabase → SQL Editor → chạy migration_payos_gateway.sql (thấy Success).
  6. Claude verify: quét QR chuyển khoản vài nghìn → app cấp gói.

Tiền về thẳng TK bạn, không phí %. Dưới 100tr/tháng chạy cá nhân; trên 100tr chuyển doanh nghiệp.

Ưu tiên 1 (thẻ quốc tế)

💳 Stripe — thẻ Visa/Master (khách du học)

  1. Tạo tài khoản dashboard.stripe.com/register → bật Test mode (góc phải).
  2. Vào /test/apikeys → copy pk_test_… và sk_test_… (bấm Reveal) → gửi Claude.
  3. Vào /test/webhooks → Add endpoint → URL: https://sat-2026.vercel.app/api/payment/stripe-webhook → chọn event checkout.session.completed → copy whsec_… → gửi Claude.
  4. Supabase → SQL Editor → chạy migration_stripe_gateway.sql (thấy Success).
  5. Claude verify: mua thử thẻ giả 4242 4242 4242 4242, hạn 12/34, CVC 123.
Ưu tiên 2

🗄️ Chạy SQL trên Supabase

Supabase → SQL Editor → New query → dán file → Run. Mỗi file 1 lần, thấy "Success".

  1. quest_claim_atomic.sql — chống nhận thưởng nhiệm vụ 2 lần.
  2. migration_exam_economy_atomic.sql — chống cộng/trừ xu sai khi nộp dồn.
  3. RLS ai_chat_cache — chặn đọc cache người khác.
  4. Cột pvp_* (nếu muốn mở PvP; bỏ qua vẫn an toàn).

Không tìm thấy file .sql? Nhờ Claude gửi nội dung để dán.

Ưu tiên 3

🔐 Đổi mật khẩu bí mật đã lộ

  1. SAT_PREP_SECRET: tạo chuỗi mới (generate-secret.vercel.app/32) → .env.local + Vercel (Sensitive) → báo Claude ký lại streak.
  2. Xóa GitHub token cũ (github.com/settings/tokens → Delete).
  3. Xóa Vercel token cũ (vercel.com/account/tokens).
  4. OpenAI key + DB password đã đổi rồi — bỏ qua.
